>> Cú pháp và lỗi ngữ nghĩa (lỗi logic)
- lỗi cú pháp xảy ra khi viết một câu lệnh không hợp lệ theo ngữ pháp của ngôn ngữ
- lỗi logic

>> chiến lược
- tìm kiếm vấn đề thông qua kiểm tra code
- tìm kiếm sự cố bằng cách chạy chương trình
- tái tạo vấn đề
- tìm hiểu về các vấn đề
Note : 
- xác thực luồng code
- in các giá trị

>> ex
- xác định hàm bị lỗi
- xác định chức năng và giá trị trả về của hàm
- lặp lại
